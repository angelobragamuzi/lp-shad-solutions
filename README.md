# lp-shad (Next.js)

Esta pasta contém a landing page convertida para um aplicativo Next.js mínimo.

Quick start

1. Instale dependências:

```bash
npm install
```

2. Rode em desenvolvimento:

```bash
npm run dev
```

3. Abra no navegador:

http://localhost:3000

Observações

- O projeto copia automaticamente a pasta `assets` para `public/assets` após `npm install` (script `postinstall`).
- Caso prefira, mova manualmente `assets/` para `public/` antes de rodar.
- Se quiser que eu gere favicons PNG (32x32 / 16x16) a partir do SVG, posso adicionar isso e atualizar as tags no `Head`.
